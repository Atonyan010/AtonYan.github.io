vultr+Ubuntu1804一键部署搭建ssr for Mac

#服务器创建#

##创建##

  • 创建新的服务器前，请确认已经充值，否则会导致无法创建成功。

##配置页面##

  • 地区选择（此处我选择的是加拿大的多伦多，根据自己需要，可选择日本、美国或其他地区，一般来说，日本速度略高，丢包率相对较高，美国速度和丢包率均可，加拿大平均，但近期，美国和日本ip被封禁概率较高，请各位自己选择）。

  • 内核系统选择（此处选择Ubuntu1804版本，原因为自带BBR加速，无需单独安装）。
  • 一般来说服务器选择最便宜的，当然大家有需要，可以选择更贵，配置更高的，此处除价格外并无其他区别。

  • 启动脚本（Startup Script）

    • 点击add new

      • name一般为固定，也可以随便起
      • type为boot

      • script内粘贴以下代码部分

        #!/bin/sh
        apt update
        apt -y install vim git python-m2crypto libsodium23
        cd /usr/local
        git clone -b manyuser https://github.com/ shadowsocksrr/shadowsocksr.git
        cd shadowsocksr
        bash initcfg.sh

    • 点击add script保存

  • SSH Keys(此处请自行百度)，或者往下看

    • 检查本机是否已有ssh key

      ls -al ~/.ssh

    • 有

    • 没有，输入以下命令

      ssh-keygen -t rsa -C “your_email@example.com“ /“此处请使用邮箱替换”

    • 然后一路回车

    • 成功后显示如下内容

      Your identification has been saved in // /.ssh/id_rsa.
      Your public key has been saved in // /.ssh/id_rsa.pub.
      The key fingerprint is:
      16:27:ac:a5:76:28:2d:36:63:1b: 56:4d:eb:df:a6:48 your_email@example.com

    • 进入~/.ssh/目录

      cd ~/.ssh/

    • 此处若报错，输入以下命令

      defaults write com.apple.finder AppleShowAllFiles -bool true /显示文件夹

      > defaults write com.apple.finder         AppleShowAllFiles -bool false /隐藏文件夹
    • 使用cat命令查看sshkey内容

      cat id_rsa.pub

    • 复制内容到正文部分（name还是随便填）

##点击Deploy Now提交保存##

  • 等待几分钟后，配置生效，服务器启动，此部分完成。
